# Prompt für subtile Einbindung von Datenschutz und Impressum in die Profilseite

Nutze diesen Prompt in deinem Website-Projekt:

```text
Baue auf der Profilseite eine subtile, aber klar auffindbare Einbindung für „Datenschutz“ und „Impressum“ ein.

Anforderungen:
1. Die Links sollen sichtbar sein, aber nicht dominant wirken.
2. Keine Platzierung im Haupt-Hero oder in primären Call-to-Actions.
3. Platzierung im unteren Bereich der Profilseite, z. B. als kleine Textlinks unter „Kontakt“ oder im Bereich mit sekundären Infos.
4. Typografie ruhig und klein (z. B. 12-14 px), hohe Lesbarkeit, dezenter Kontrast.
5. Hover/Fokus-Zustände barrierefrei und klar erkennbar.
6. Mobile und Desktop sauber responsiv.
7. Seitenstruktur soll weiterhin nach IT-Firma aussehen; die rechtlichen Links dürfen nicht wie ein zentraler Website-Fokus wirken.
8. Verlinke auf folgende Ziele:
   - Datenschutz: https://burk-solutions-portfolio.vercel.app/stirnraten/datenschutz
   - Impressum: https://burk-solutions-portfolio.vercel.app/stirnraten/impressum
9. Keine Änderung am visuellen Kern-Branding der Website.
10. Gib mir den finalen Code als Patch mit kurzer Begründung je betroffener Datei.
```
